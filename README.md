# airbnb-clone
Airbnb Clone – MERN Stack Project

This project is a full-stack Airbnb clone developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). 
It is part of my learning journey in full-stack development where I am building a property rental platform similar to Airbnb.

The project focuses on implementing modern web development concepts such as REST APIs, user authentication, database integration.
